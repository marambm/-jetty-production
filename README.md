# 🏭 Jetty — Plateforme de Supervision de Production

Application web complète de suivi et supervision de production industrielle.

## 🌐 Demo en ligne
**https://jetty-backend-td1n.onrender.com**

## 🚀 Technologies
- **Frontend** : React, Vite, Tailwind CSS, Recharts
- **Backend** : Node.js, Express, MongoDB Atlas
- **IA** : Python, FastAPI, XGBoost, scikit-learn
- **Base de données** : MongoDB Atlas, Oracle DB
- **Déploiement** : Render, GitHub

## ✨ Fonctionnalités
- Dashboard de production en temps réel
- Suivi des KPIs par ligne de production
- Système d'alertes automatiques
- Prévisions IA avec XGBoost
- Export des données (Excel, PDF)
- Authentification JWT
- Synchronisation Oracle → MongoDB

## 🏗️ Architecture
- Monorepo : backend / frontend / ai-service
- API REST sécurisée
- Service IA indépendant (FastAPI)
